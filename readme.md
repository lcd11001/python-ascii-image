# Convert Image to ASCII

- This is a simple python script that converts an image to ASCII art. 
- The script uses the Python OpenCV Library (CV2) and Numpy to open and manipulate images.
- The script reads the image pixel by pixel and converts the RGB values to a grayscale value. 
- The grayscale value is then mapped to an ASCII character based on its intensity. 
- The script outputs the ASCII art to the console.
